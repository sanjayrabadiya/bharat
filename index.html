    Printed Labels

    We give multicolour printed labels on various substrates like Paper, Polyster, BOPP, Mat film etc. For various application in the industries like Pharma, FMCG, Cosmetic industry etc. All our printed labels can have special features like Hologram, Foiling, Emboissing and security marks.

    Speciality & Security Labels

    Duralabel is considered to be one of the best in Asia for speciality, Security & Anti counterfeiting Labels. We cater to various kinds of security Labels like Tamper Evident Labels, Security VOID Labels, Tyre Labels, Heat Resistant Labels, Hi Glue Labels, Removable Labels.

    Shrink Sleeves

    Our shrink sleeves are crafted using state-of-the-art printing techniques and high-quality polymer films to ensure exceptional print quality, durability, and resistance to moisture, scratching, and fading.
